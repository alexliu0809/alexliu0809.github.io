HitPredictor

Project Introduction

Warner Music wants to sign any talent that attracts a sizeable audience, from gaming, to comedy, to make-up tutorials. As specified by our clients, this system is designed to monitor new YouTube channels and flags hot and upcoming talent to talent scouts.

Key Features
  • HTML5/CSS3 Front-end for presenting hot and upcoming artists
  • Dynamic database to keep track of channel nodes, make API requests to different social media networks at a set interval and update information in database.
  • Sentiment analysis for Twitter account linked with YouTube channel
