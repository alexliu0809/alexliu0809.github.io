Geoda

Project Introduction

GeoDa is a free and open source software tool that serves as an introduction to spatial data analysis. The program provides a user-friendly and graphical interface to methods of exploratory spatial data analysis, such as spatial autocorrelation statistics, and basic spatial regression analysis for lattice data.

Key Features
  • Multiple views on the data
  • Dynamic linking and brushing -- observations selected in one view are immediately selected in all views
  • Assessing spatial heterogeneity -- structural breaks across space
  • Space-time exploration
  • Other features including: data entry, data manipulation, mapping and geovisualization, built-in regression functions, etc..
